## Using gulp for builds (FE)
1. go to theme folder ```cd wp-content\themes\bestcasino```
2. ```npm install```
3. Run the build:
-  for development (files are not minified)
```npm run watch``` or ```npm run gulp```

-  for production
   ```npm run build``` or ```npm run prod```
